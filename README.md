# Автоматизация теста к API
- Создание заказа и проверка, что его можно получить по трек‑номеру.
## Какие файлы есть
- configuration.py - URL и API
- sender_stand_request.py - Cам тест: создаёт заказ, берёт трек, запрашивает заказ по треку, сверяет трек и статус.
- data.py — данные : шаблон тела запроса для создания заказа.
- Скрин SQL запроса из терминала : Вывод списка логинов курьеров с количеством их заказов в статусе "В доставке"
- Скрин SQL запроса из терминала : Вывод всех трекеров заказов и их статусы.
- Скриншот с запуском теста в Visual Studio
